Bio

Coding (or trying to), building my first websites, tinkering with components and of course playing games. In 2020, Covid brought work to a stillstand there which gave me the time to reevaluate and decide that my desire for coding had not been fulfilled yet. Quickly I found neuefische's Web Development bootcamp which is currently bringing me up to speed in the MERN stack.
